# 专注时长统计 - 使用说明

> 校园AI学习助手的专注计时模块

## 功能特性

- ⏱️ **专注计时器** - 灵活的专注时长选择（15/25/45/60分钟）
- 🚫 **防干扰提醒** - 检测页面切换并温馨提醒
- 📊 **数据统计** - 今日/本周专注时长可视化
- 📈 **趋势图表** - 本周一览无余
- 💾 **本地存储** - LocalStorage 自动保存，无需注册

## 使用方法

### 直接使用（推荐）

```
直接双击打开 index.html 文件即可使用
```

### 本地服务器

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

然后访问 http://localhost:8080

## 功能说明

### 1. 开始专注
- 选择目标时长（15/25/45/60分钟）
- 点击「开始专注」按钮
- 计时器开始倒计时

### 2. 专注模式
- 专注期间切换标签页会触发提醒
- 显示离开次数，但不强制中断
- 随时可点击「结束专注」

### 3. 数据统计
- **今日专注** - 当天累计专注时长
- **本周累计** - 本周所有专注时长总和
- **专注次数** - 今日完成的专注次数
- **本周趋势** - 柱状图展示每日专注情况

### 4. 历史记录
- 显示最近5条专注记录
- 包含日期、时间和时长
- 支持清除所有记录

## 快捷键

| 按键 | 功能 |
|------|------|
| 空格键 | 开始/结束专注 |

## 数据存储

所有数据保存在浏览器 LocalStorage 中：
- 无需注册登录
- 关闭页面不丢失
- 清除浏览器会重置数据

如需备份数据，可打开浏览器开发者工具（F12）：
```javascript
// 导出数据
JSON.parse(localStorage.getItem('focus_timer_data'))

// 清除数据
localStorage.removeItem('focus_timer_data')
```
